Hacerca de mi

No soy un diseñador de mente intuitiva o tan creativo si tubiera una clasificación en cuanto al tipo de diseñaor que soy yo creo que seira uno el cual elabora los diseños enbase alos elementos que proporciona el cliente para poder hacer los anuncios que este necesite un diseñador elabora el un diseño el cual este comunica para poder resolver un problema el cliente proporciona los productos nosotros ofresemos la solucion lo que pide pue ser el anuncio de un nuevo producto, una promoción, la apertura de un negocio etc pero en una realidad laboral los clientes tienen una idea espesifica de lo que quieren y aun que les presentemos las opciones que les ayudaria para aumentar sus ventas estos mismos clientes demuestran tener una idea indomable o como lo llamo espiritu de ballata por la cual luchan y piden alos diseñadores que se les respete su idea y que esta sea representada de la manera que ellos quieren bueno ante estos casos como sedice se hace como el cliente pida pero tambien es nuestro deber eñ ayudar al cliente por lo que mejoramos un poco su idea para que sea con un impacto visual mayor esto es alo que yo considero la elaboracion del diseño enbase alos elementos dados por el cliente.
